# Emuladores SuperTEF

Desenvolva e integre sua aplicação ao **SuperTEF** usando nossos emuladores para **Windows** e **Android**.

Com o emulador, você não precisa possuir uma máquina SmartPOS física para iniciar sua integração. Basta instalar, configurar sua aplicação e começar a testar pagamentos, consultas de status e estornos diretamente no ambiente de desenvolvimento.

## Integração simplificada

Se sua aplicação integrou e funcionou 100% no emulador, então ela já está preparada para operar com todos os bancos e adquirentes compatíveis com o SuperTEF.

## O que é possível testar

- Pagamentos
- Consulta de status
- Estornos
- Fluxos de retorno
- Comunicação com o ecossistema SuperTEF
- Homologação da integração

## Plataformas

- Windows
- Android

## Por que usar o emulador?

O emulador SuperTEF permite que desenvolvedores criem, testem e validem suas integrações sem depender de equipamentos físicos, reduzindo tempo, custo e complexidade no processo de desenvolvimento.

## Mensagem principal

Funcionou no emulador?  
Então sua integração está pronta para o ecossistema SuperTEF.
